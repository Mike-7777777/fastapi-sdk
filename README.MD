`智元 Fast API` 是一站式API管理系统，将各类大模型API进行统一格式、统一规范、统一管理，使其在功能、性能和用户体验上达到极致。

## 功能介绍
- 智元 Fast API SDK模块, 主要集成各类 `大模型 API` 的能力
